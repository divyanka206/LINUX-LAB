## β…How the script works
- The script ensures it runs from its own directory.
- Creates a `backup/` folder (if missing).
- Finds all `.txt` files in the current folder.
- Copies each `.txt` into `backup/` and appends a timestamp `YYYYMMDD_HHMMSS` to the filename.

## π“Commands:

### π”ΉCreating script for backup (backup.sh)
```bash
touch backup.sh
```

### π”ΉCreating files
```bash
touch file1.txt
touch file2.txt
```

### π”Ήecho command
```bash
echo "hello world" > file1.txt
```

### π”Ήchmod command
```bash
chmod 777 backup.sh
```

### π”ΉRun backup script
```bash
./backup.sh
```
